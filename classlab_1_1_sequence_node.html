Detailed Description

One of the most important flow nodes. If one of child nodes returns false after run, SequenceNode also will return false.

SequenceNode aggregates child nodes and runs them from first to last. It stops running after first child node run result that is false, that means not all child nodes must be invoked. If all child nodes returns true, this node will also return true as a run result.
