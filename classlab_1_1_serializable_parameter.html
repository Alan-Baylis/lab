Detailed Description

Generic class for dictionary serialization.

Unity cannot serialize dictionaries, so SerializableParameter implements ISerializationCallbackReceiver. This gives possibility of custom serialization. However, this solition does not work with generic types, so we need to inherit from SerializableParameter.
